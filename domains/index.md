# Assessment Domains

STEP (Strengthening and Tiered Evaluation Process) assesses civil society organizations across a set of domain areas — referred to as "streams" in TechSoup materials. Each domain is assessed and scored independently, meaning a CSO receives a tier rating per domain rather than a single org-wide score.

## How Criteria Are Structured

Each domain's `criteria.md` file contains requirements organized by tier:

- **Tier 1 — Basic**: Minimum requirements for grassroots, early-stage organizations
- **Tier 2 — Foundation**: Formal structures and documented processes
- **Tier 3 — Agency**: Mature internal systems and comprehensive policies
- **Tier 4 — Plus**: Highest level of organizational capacity

Tiers are additive within each domain — Tier 2 includes all Tier 1 requirements plus additional criteria. See the [Tiered Framework Design](../docs/tiered-framework) for full design rationale.
